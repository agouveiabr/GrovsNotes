# GrovsNotes: Advanced Capture & Navigation

The primary mission of GrovsNotes is to provide a high-performance, developer-centric task management environment. This project focus is on **Advanced Capture & Navigation**, enabling frictionless task entry through natural language parsing and rapid keyboard-first navigation.

## Core Value

Frictionless capture—the ability to turn a thought into a structured task with metadata (tags, projects, priority, dates) in seconds without leaving the keyboard.

## Key Goals

1. **Intelligent Capture**: Automatically extract metadata from raw input text to reduce manual categorization.
2. **Speed-First Navigation**: Allow users to navigate and perform actions across the entire application without using a mouse.
3. **Trust-Building Feedback**: Provide immediate visual confirmation of how the system interpreted user input.

## Success Metrics

- **Capture Speed**: Users can create a task with 3+ pieces of metadata in under 5 seconds.
- **Mouse Reliance**: Users can complete 90%+ of common workflows (navigation, search, creation) exclusively via keyboard.
- **Parsing Accuracy**: High trust in the NLP engine's ability to extract tags and dates correctly.

## Constraints

- **Web-First**: Primary focus is the web application experience.
- **Low Latency**: UI feedback (chipping, highlighting) must happen in real-time without perceived lag.
- **Data Integrity**: Never lose the user's original input, even if the parser fails to extract metadata.
